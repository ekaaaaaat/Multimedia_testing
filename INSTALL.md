# Инструкция по установке

## Шаг 1: Установка Node.js

Для работы проекта необходимо установить Node.js (версия 16 или выше).

1. Скачайте Node.js с официального сайта: https://nodejs.org/
2. Установите Node.js (при установке npm будет установлен автоматически)
3. Проверьте установку, выполнив в командной строке:
   ```
   node --version
   npm --version
   ```

## Шаг 2: Установка зависимостей проекта

После установки Node.js, откройте терминал в папке проекта и выполните:

```bash
npm install
```

Эта команда установит все необходимые зависимости:
- React 18
- Vite
- Плагины для работы с React

## Шаг 3: Запуск проекта

После установки зависимостей запустите проект:

```bash
npm run dev
```

Проект будет доступен по адресу: http://localhost:3000

## Альтернативный способ установки

Если у вас проблемы с npm, вы можете использовать:
- **yarn**: `yarn install` и `yarn dev`
- **pnpm**: `pnpm install` и `pnpm dev`
